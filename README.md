# Welcome to the SU air 

## Repository Naming Convention
Suggested structure for our repositories:
`<category>-<topic>-<year>`

### Categories
- `education` – courses, lectures, assignments  
- `modelling` – numerical models, simulations, ML work  
- `experiment` – lab work, instrumentation  
- `infrastructure` – reading functions, other templates  

### Examples
- `education-course-ACP-2025`
- `modelling-miecode-2025`
- `experiment-SEMS-calibration-2025`
- `infrastructure-read-CVI`

## Folder Structure
- `docs/` – documentation  
- `src/` – code  
- `notebooks/` – Jupyter notebooks  
- `tests/` – tests
